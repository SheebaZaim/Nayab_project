# Nayab Project Backend

A Python Flask backend for the Nayab e-commerce platform specializing in embroidered clothing and accessories.

## Features

- User authentication and authorization
- Product catalog management
- Shopping cart functionality
- Order processing
- Admin dashboard
- Category management
- Review system

## Tech Stack

- Python 3.9+
- Flask web framework
- SQLAlchemy ORM
- Flask-Login for authentication
- SQLite database (configurable for PostgreSQL in production)
- Jinja2 templates (for admin views)
